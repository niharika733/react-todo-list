# To-Do List React App

This is a simple single-page To-Do List application built using **React** and **Vite**.

## Features

* Add tasks using an input field
* Display tasks in a list
* Mark tasks as completed using checkboxes
* Delete tasks from the list
* No database (uses React state only)

## Tech Stack

* React (JavaScript)
* Vite
* Basic CSS

## How to Run Locally

* All the files are in src and index.html is seperately present

```bash
npm install
npm run dev
